WordPress Dashboard Credits Removal Removes all WordPress credits from dashboard


== Description ==


WordPress Dashboard Credits Removal is perfect if you run a client website. It hides all WordPress credits from the dashboard section of your site.


== Installation ==


1. Upload `WPDCR.php` to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. In your theme's image file upload an image and call it logo_admin.png this is important otherwise the WordPress logo on the login page will not be replaced!
